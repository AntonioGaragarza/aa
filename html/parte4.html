Ranuras de la tarjeta madre

Slot o ranura de expancion:

Es un componente en la tarjeta madre que permite conectar tarjetas adicionales o de expancion,
para funiones para contro a dispocitivos perifericos adicionales,como monitor,impresoras o puertos USB


  1. AGP(Accelerated Grafios Port):
    Usado para tarjetas de graficos o videos

  2. PCI(Peripheal Componet Interconet)
    Agregar tarjetas de red,puertos USB,t. de sonido,etc.

  3. ISA(Industry Standar Arclitecture)
    Agregar tarjetas de red,video,etc. con tecnologia IRM

  4. DDR(Double Data Rate)
    Ranuras para memoria RAM

  5. IDE(Integrated Development Envirument)
    Interfas para conexion de dispocitivos de alamcenamiento

  6. SATA(Serial Advanced Tecnologi Attachment)
    Interfas para bus de datos para la transferiencia de datos entre persona base y dispositivos de alamcenamiento
